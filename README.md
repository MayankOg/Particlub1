## Getting Started ğŸ

### Prerequisites

- Flutter SDK
- Amplify CLI
- Android Studio or Xcode

### Installation

1. **Clone the Repository**
   ```bash
   git clone git@github.com:MayankOg/Particlub.git
   cd Particlub
   ```

2. **Install Dependencies**
   ```bash

   ```


   ```bash
   flutter run
   ```
